Me and Machine Learning

Why am I doing machine learning?

I started my journey into machine learning in December of 2013. It has been difficult for me. Thus far, my favorite resources have been MITx's The Analytics Edge, Dr. Andrew Ng's machine learning courses (Stanford, Coursera), and the book An Introduction to Statistical Learning with Applications in R. However, I only actually completed the first one. I have tried so many resources and have been unable to get a strong enough grasp on machine leaning in order to understand how to use it. Most resources have either been too focused on application and a little theory or too focused on theory and a little on the applications. I have not been able to find the right balance, but I really want to do machine learning. I want to learn from data because of the potential ability to improve the lives of others and transform organizations' productivity. I find the idea of making fairly accurate predictions on uncertain outcomes using data exciting. I would like to have the ability to be depended upon to analyze data to answer questions that an organization may have.

How will I continue developing my machine learning journey?

Everytime I begin using a resource to develop my machine learning skills, I feel that I have finally found the right resource for me. I turn out to be wrong, but I feel that I can now break that barrier. I will now be enrolling in Coursera's Machine Learning Specialization and using the book Mastering Predictive Analytics with R.
